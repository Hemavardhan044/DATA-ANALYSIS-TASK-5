# Task 5: Data Analysis on CSV Files Using Jupyter Notebook

## Objective
Perform basic data analysis and visualization on CSV data using Python, Pandas, and Matplotlib in a Jupyter Notebook.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Description
This notebook demonstrates how to:
- Create sample CSV data directly within the notebook
- Load CSV data into a Pandas DataFrame
- Explore the dataset using basic statistics
- Perform grouping and aggregation operations
- Visualize sales data using charts

## Steps Performed

### 1. Create Sample CSV Data
A sample sales dataset is created inside the notebook using a multi-line string.

### 2. Load Data
The CSV data is loaded into a Pandas DataFrame using `pd.read_csv()`.

### 3. Data Exploration
The notebook displays:
- First 5 rows of the dataset
- Dataset shape
- Summary statistics

### 4. Grouping and Aggregation
Sales data is analyzed by:
- Product-wise total sales
- Month-wise total sales

### 5. Data Visualization
Two charts are generated:
- **Bar Chart**: Total Sales by Product
- **Line Chart**: Monthly Sales Trend

## How to Run

1. Open Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

2. Create a new notebook and paste the code into a cell.

3. Run the cell to view:
   - Data analysis results
   - Sales summaries
   - Visualizations

## Output
The notebook displays:
- Dataset information
- Product-wise sales totals
- Monthly sales trends
- Bar chart and line chart visualizations

## Learning Outcomes
- Working with CSV data in Jupyter Notebook
- Using Pandas for data analysis
- Applying grouping and aggregation
- Creating charts with Matplotlib
- Understanding sales trends through visualization
